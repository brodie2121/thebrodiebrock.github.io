Skills/My Projects

TindCodr

TindCodr is a web application which allows junior and senior developers to connect with one another in their geographic area. Not only does a user have the ability to connect with other users, they can also add projects, comment on other user’s projects, and collaborate with other users on new projects. The user can sign up with their slack account or sign up by email. TindCodr is a back-end based project using NodeJS, SQL, Express, Javascript, HTML, and CSS.

Road Rage

Road Rage is a video game that involves the user to dash in and out of traffic to gain points. The game is made from the computer program pygame that uses a set of Python modules to design the game.
